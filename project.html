My Notes

·Android project

It is an application that helps to keep notes .
The app uses java ,xml and Room database.
User can create ,read ,update and delete notes.
Each note has a title and day, date and time on them when it they were created and if they they are updated day ,date and time changes to updated ones. there is a different color and each time the application is opened the colors change .
There is also a speech to text function added for creating notes.
No empty notes can be created and saved.

Game-Galaxy

·Python game project

The game is named "Galaxy" and is made using python and specifically "kivy library".
It can be played in both 2D and 3D.
There is a triangle shaped spaceship which has to be on the track that is produced randomly for the user .
After a certain score the player moves to next level and the screen showing entering next level is displayed and the speed of the spaceship increases as the level increases.
When the game is over the player is asked if he wants to restart it.
Both the best score and score are displayed on the screen and there are various audios in the game at start ,end while playing and restarting.
